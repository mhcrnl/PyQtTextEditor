# PyQtTextEditor

Based on QtCreator 4.12.4 (with Qt 5.14.2)

## Install

	git clone https://github.com/mhcrnl/PyQtTextEditor.git
	cd PyQtTextEditor
	python3 text_editor.py

## or

	QtCreator project
